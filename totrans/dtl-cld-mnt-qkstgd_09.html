Section 2: Extending Datadog

Datadog has many standard monitoring features that are out of the box and immediately available upon successfully installing agents in an application environment. However, to roll out a comprehensive monitoring solution, Datadog needs to be extended by adding custom metrics and features. The Datadog platform provides multiple integration options to extend it to implement advanced monitoring requirements. This part of the book explores all such options in detail.

This section comprises the following chapters:

  • Chapter 8, Integrating with Platform Components
  • Chapter 9, Using the Datadog REST API
  • Chapter 10, Working with Monitoring Standards
  • Chapter 11, Integrating with Datadog
